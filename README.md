# 🏦 Bank Management System

A simple yet functional Bank Management System developed using Python and Streamlit. This project demonstrates the practical implementation of Object-Oriented Programming (OOP), file handling, JSON data storage, and a graphical user interface using Streamlit.

The application allows users to perform common banking operations through an interactive web interface while storing all account information locally in a JSON file.

## Features

- Create a new bank account
- Automatically generate a unique account number
- Deposit money
- Withdraw money
- Check account balance
- Transfer money between accounts
- Update account information
- Delete an account
- Persistent data storage using JSON
- User-friendly Streamlit interface

## Technologies Used

- Python
- Streamlit
- Object-Oriented Programming (OOP)
- JSON
- File Handling
- Random Module
- Pathlib
